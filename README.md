# serv00-auto-login-ssh
serv00 ssh 自动登录一下啦

## 环境变量
|名称|作用|
|-|-|
|ID_RSA|ssh sra私钥，用于登录到服务器|
|USER|ssh用户名|
|SERVER|ssh密码|
